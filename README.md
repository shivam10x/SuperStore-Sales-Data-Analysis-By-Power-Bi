#  Superstore Data Analysis Dashboard

A visually engaging and insightful data analysis project built using Superstore sales data. This dashboard uncovers key business insights across sales, profit, customer segments, shipping modes, and regional performance.

---

## 📊 Project Overview

The **Superstore Data Analysis Dashboard** provides a comprehensive view of retail performance through interactive and easy-to-understand visualizations. It helps identify trends, monitor KPIs, and support data-driven decision-making.

---

## 🚀 Key Highlights

* 💰 **Total Sales:** 1.57M
* 📈 **Total Profit:** 175.26K
* 📦 **Total Quantity Sold:** 22K
* ⏱️ **Average Delivery Time:** 4 Days
* 🔁 **Returns:** 287 Orders

---

## 📌 Features

* 📍 **Region-wise Filtering** (Central, East, South, West)
* 📅 **Monthly Sales Trends** (Year-wise comparison)
* 🧾 **Sales by Category & Sub-Category**
* 🚚 **Shipping Mode Analysis**
* 💳 **Payment Mode Distribution**
* 👥 **Customer Segmentation Insights**
* 🗺️ **Geographical Sales Visualization (Map View)**

---

## 📊 Insights Derived

* **COD** is the most preferred payment method (~43%)
* **Consumer segment** contributes nearly half of total sales
* **Office Supplies** lead in category sales
* **Standard Class shipping** dominates delivery mode
* Seasonal spikes observed in **October–December sales**
